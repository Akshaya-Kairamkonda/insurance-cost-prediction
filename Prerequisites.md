 ## Pre-requisites

### Software Requirements

* Python 3.7 or above
* Jupyter Notebook / VS Code / PyCharm

### Python Libraries

* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

### Installation

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

### Hardware Requirements

#### Minimum

* 4 GB RAM
* Dual-Core Processor
* 500 MB Free Disk Space

#### Recommended

* 8 GB RAM or higher
* Intel i5 / Ryzen 5 or equivalent
* SSD Storage

### Dataset

* insurance.csv
* Contains information such as age, gender, BMI, number of children, smoking status, region, and medical insurance charges.

### Machine Learning Algorithm

* Linear Regression

### Development Environment

* Jupyter Notebook (.ipynb)
* Python Script (.py)
